## 발견된 이슈

### ISSUE-1: `<a>` 태그 중첩 hydration 에러 (해결 완료)

- **증상**: 피드 메인 페이지 로드 시 콘솔에 "In HTML, `<a>` cannot be a descendant of `<a>`" hydration 에러 2건
- **원인**: `FeedList`에서 카드 전체를 `<Link>`로 감싸고 있는데, `FeedCard` 내부 댓글 아이콘도 `<Link>`여서 `<a>` 안의 `<a>` 중첩 발생
- **수정**: `FeedCard.tsx`의 댓글 아이콘을 `<Link>` → `<span>`으로 변경 (카드 클릭 시 상세 이동이므로 별도 링크 불필요)
- **상태**: **해결 완료** — 수정 후 콘솔 에러 0건 확인

### ISSUE-2: Next.js 라우트 그룹 충돌 (해결 완료)

- **증상**: `/feed` 접근 시 "You cannot have two parallel pages that resolve to the same path. Please check /(auth)/feed and /(public)" 빌드 에러
- **원인**: `(auth)/feed/write`와 `(public)/feed`가 동일 `/feed` 경로로 해석되면서 충돌
- **수정**: `(auth)/feed` 디렉토리 삭제, 작성 페이지를 `(public)/feed/write`로 이동 + 페이지 내부에서 인증 가드 직접 처리
- **상태**: **해결 완료**
