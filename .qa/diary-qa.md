# Diary 도메인 QA 테스트 시나리오

> 대상 범위: 일기 CRUD + 태그 입력 + 태그 자동완성
> 프론트엔드 페이지: 일기 목록, 작성, 상세, 수정
> 백엔드 전제: `POST/GET/PUT/DELETE /api/v1/diaries`, `GET /api/v1/tags/autocomplete` 정상 동작

## 테스트 환경 조건

- **백엔드**: `http://localhost:8080` 기동 필수
- **인증**: 테스트 계정으로 로그인 상태
- **브라우저**: Chrome 최신, 모바일 뷰포트(375px) + 데스크탑(1280px) 양쪽 확인
- **네트워크 탭**: API 요청/응답 확인 (특히 `withCredentials`, Cookie 전송 여부)
